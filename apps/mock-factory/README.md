# 🏭 Mock Factory

> 개발 환경에서 필요한 더미(Mock) 데이터를 쉽고 직관적으로 생성할 수 있는 웹 기반 도구입니다. 복잡한 설정 없이 UI를 통해 원하는 데이터 스키마를 구성하고, 즉시 JSON 결과물을 확인 및 내보낼 수 있습니다.

## ✨ 주요 기능

- **직관적인 스키마 에디터**: 클릭 몇 번으로 원하는 데이터 필드를 추가하고 구성할 수 있습니다.
- **다양한 데이터 타입 지원**: `@faker-js/faker`를 기반으로 이름, 이메일, 연락처, 주소, UUID 등 실무에 가까운 더미 데이터를 생성합니다.
- **커스텀 제어**:
  - `일련번호`: 접두사 및 시작 번호 지정 가능
  - `무작위 선택`: 사용자가 원하는 옵션들을 콤마(,)로 구분하여 입력하면 그 중 하나를 무작위로 반환
- **유연한 출력 형식**: 단일 객체(Object) 또는 지정된 개수만큼의 배열(List) 형태로 결과를 생성합니다.
- **개발자 친화적 내보내기**:
  - **Copy JSON**: 생성된 데이터를 클립보드에 복사합니다.
  - **Download**: `.json` 파일로 즉시 다운로드합니다.

## 🛠 기술 스택

- **Framework**: React 19, Vite 8
- **Language**: TypeScript 5.9
- **Styling**: Tailwind CSS v4
- **Core Library**: `@faker-js/faker` v10

## 🚀 설치 및 실행

프로젝트를 로컬에서 실행하려면 아래 명령어들을 순서대로 입력하세요.

```bash
# 1. 패키지 설치
npm install
# 또는 yarn install / pnpm install

# 2. 개발 서버 실행
npm run dev
# 또는 yarn dev / pnpm dev
```
